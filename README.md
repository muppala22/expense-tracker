A simple expense management SPA built with Angular standalone components, routing, HttpClient, and Angular Material.

## Features
- List, add, edit, delete expenses
- REST API integration (json-server)
- Reactive forms with validation
- Reusable components and services
- Responsive UI with Angular Material
- Deployed on Firebase

## Tech Stack
- Angular 17 (standalone)
- TypeScript
- Angular Material
- RxJS
- JSON Server (mock backend)
- Firebase Hosting

## How to Run
1. Clone:
   git clone https://github.com/yourusername/expense-tracker

2. Install dependencies:
   npm install

3. Start mock backend:
   json-server --watch db.json --port 3000

4. Start Angular app:
   ng serve -o
